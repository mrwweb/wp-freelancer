Assist WordPress Freelancers to manage a client site.

== Description ==

Assist WordPress Freelancers to effectively manage a client site. Also helps to 
protect clients who are unfamiliar with WordPress by using a new admin role.


== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ 
directory of your WordPress installation and then activate the Plugin from 
Plugins page.
